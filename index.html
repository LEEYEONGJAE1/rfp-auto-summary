Gemini API 키 발급 방법

Google 계정만 있으면 2분이면 됩니다. 신용카드 불필요·무료예요. 아래 버튼으로 발급 페이지를 연 뒤 순서대로 따라 하세요.

발급 페이지 열기 ↗
  1. Google AI Studio 발급 페이지 접속위 "발급 페이지 열기" 버튼을 누르고, Google 계정으로 로그인합니다. (회사 또는 개인 계정 모두 가능)
  2. "API 키 만들기"(Create API key) 클릭화면 가운데 또는 우측의 파란 버튼입니다.
  3. 프로젝트 선택창이 뜨면 "새 프로젝트에서 API 키 만들기"를 고릅니다. (기존 프로젝트가 있으면 그것을 선택해도 됩니다.)
  4. 키 복사잠시 후 AIza… 로 시작하는 키가 생성됩니다. 옆의 복사 아이콘을 눌러 복사합니다.
  5. 이 화면에 붙여넣기왼쪽 "Gemini API 키" 칸에 붙여넣으면 끝. "모델 불러오기"로 모델 목록까지 확인하면 준비 완료입니다.
API 키는 비밀번호처럼 다뤄 주세요. 외부에 공유하거나 화면에 노출하지 않는 게 좋습니다. 이 도구는 키를 어떤 서버에도 보내지 않고 이 브라우저에서만 사용합니다.
무료 등급은 분당·일일 요청 한도가 있어, 많이 막히면 잠시 뒤(또는 한도 리셋 후) "추출 시작"을 다시 누르면 남은 것부터 이어집니다.
